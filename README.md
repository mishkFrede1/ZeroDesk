## Overview
ZeroDesk is an AI-powered media platform that automatically collects news from multiple sources, analyzes them using NLP models, and publishes structured articles with headlines, tags, and categories. The system combines a microservices architecture with real-time data processing to deliver fresh and reliable news for readers around the world.
